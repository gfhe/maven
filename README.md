# maven 仓库

项目中增加如下的repository:

```
<repository>
    <id>heguangfu's maven</id>
    <url>https://mvn.hgfdodo.win</url>
</repository>
```

依赖使用方法， 以 `minio-spring-boot-starter` 为例：

```xml
<dependency>
    <groupId>win.hgfdodo</groupId>
    <artifactId>minio-spring-boot-starter</artifactId>
    <version>1.0</version>
</dependency>
```
